데이터 분석 실습 (3차시 구성)

💡 수업 철학 (파인만 기법)

“진짜 이해했다면 쉽게 설명할 수 있다.”

  • 초등학생도 이해할 수 있게 설명
  • 어려운 용어는 풀어서 이야기
  • 예시, 비유 활용
  • 학생이 직접 설명하게 유도
  • 내가 모르는 건 학생도 어렵다고 가정하고 설명

✅ 전체 차시 개요

차시 주제 목표
1차시 데이터 분석이란? 데이터 분석 개념 이해, 숫자 → 이야기 전환 이해
2차시 데이터를 정리하고 시각화하기 표와 그래프 만들기 실습
3차시 데이터에서 인사이트 찾기 표/그래프 해석 → 스토리 만들기

🗂️ 차시별 지도안

수업 목표

  • 데이터 분석의 의미를 쉽게 설명할 수 있다.
  • 데이터를 보면 이야기가 숨겨져 있다는 걸 이해한다.

수업 흐름

  1. 도입 (5분): 질문 나누기
  2. 전개1 (10분): 파인만식 개념 설명
  3. 전개2 (15분): 소그룹 활동
  4. 정리 (5분): 오늘 내용 되새기기

파인만식 한줄 요약

“데이터 분석은 숫자에서 이야기를 찾는 거야.”

수업 목표

  • 데이터를 표로 정리할 수 있다.
  • 그래프로 시각화할 수 있다.

수업 흐름

  1. 도입 (5분): 지난 시간 복습
  2. 전개1 (10분): 표 만들기 시연/실습
  3. 전개2 (15분): 그래프 만들기 시연/실습
  4. 정리 (5분): 발표 및 공유

파인만식 한줄 요약

“표와 그래프는 데이터를 쉽게 읽게 해주는 이야기 책이야.”

수업 목표

  • 데이터를 보고 중요한 정보를 찾을 수 있다.
  • 데이터를 친구에게 쉽게 설명할 수 있다.
  • 그래프를 해석하고 스토리를 만들 수 있다.

수업 흐름

  1. 도입 (10분): 지난 시간 복습 및 목표 안내
  2. 전개1 (10분): 파인만식 설명과 예시
  3. 전개2 (15분): 소그룹 실습 및 발표 준비
  4. 정리 (5분): 발표와 피드백
